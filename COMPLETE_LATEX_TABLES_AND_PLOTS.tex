\chapter*{Executive Summary}
\addcontentsline{toc}{chapter}{Executive Summary}

This comprehensive document presents the complete analysis results from the Register Comparison study between canonical and headline language varieties across three major Indian English newspapers: \textbf{Times of India}, \textbf{The Hindu}, and \textbf{Hindustan Times}.

\section*{Analysis Scope}
\begin{itemize}
    \item \textbf{Data Volume}: 227 analytical tables and 228 visualization plots
    \item \textbf{Newspapers}: 3 major Indian English publications
    \item \textbf{Linguistic Features}: 18 morpho-syntactic transformation types
    \item \textbf{Analysis Types}: Comprehensive, Cross-entropy, Feature-value, Statistical
    \item \textbf{Methodologies}: Information-theoretic measures, Statistical analysis, Visual analytics
\end{itemize}

\section*{Key Findings Summary}
\begin{itemize}
    \item \textbf{Bidirectional Cross-Entropy}: Quantified information loss between registers
    \item \textbf{Feature Analysis}: 88 detailed feature-value transformation tables
    \item \textbf{Register Differences}: Systematic patterns across newspapers
    \item \textbf{Information Asymmetry}: Directional differences in register complexity
\end{itemize}

\section*{Document Organization}
This document is organized into the following major sections:
\begin{enumerate}
    \item \textbf{Comprehensive Analysis Tables} (20 tables)
    \item \textbf{Bidirectional Cross-Entropy Analysis} (26 tables)
    \item \textbf{Feature-Value Analysis} (88 tables)
    \item \textbf{Feature-Value Pair Analysis} (15 tables)
    \item \textbf{Statistical Summary Tables} (9 tables)
    \item \textbf{Newspaper-Specific Analysis} (8 tables)
    \item \textbf{TED Analysis Tables} (4 tables)
    \item \textbf{Visualization Plots} (228 figures)
\end{enumerate}

\section{Data Processing Pipeline}

The complete analysis pipeline processes 227 CSV data tables and generates 228 visualization plots through the following stages:

\begin{enumerate}
    \item \textbf{Data Extraction}: Linguistic features extracted from parsed sentence pairs
    \item \textbf{Feature Analysis}: Individual feature transformations analyzed
    \item \textbf{Cross-Entropy Calculation}: Bidirectional information-theoretic measures computed
    \item \textbf{Statistical Analysis}: Comprehensive statistical summaries generated
    \item \textbf{Visualization Generation}: Multiple plot types created with mnemonic labels
\end{enumerate}

%-------------------------------------------------------------------
\section{Methodology Notes}

\begin{itemize}
    \item \textbf{Cross-Entropy}: Measured in bits using log base 2
    \item \textbf{Jensen-Shannon Divergence}: Symmetric measure bounded [0, 1]
    \item \textbf{Information Asymmetry}: |CE(Canonical→Headlines) - CE(Headlines→Canonical)|
    \item \textbf{Register Overlap Ratio}: Shared values / Total unique values
    \item \textbf{Feature Labels}: Mnemonic codes used throughout for readability
\end{itemize}
